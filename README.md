# l_p_solutions_feladat

This is a test project for an application to L&P solutions company

## Installation & run
- cd into `/demo` folder
- Install dependencies
  - run `mvn clean install -DskipTests`
- start the server
  - java -jar .\target\demo-0.0.1-SNAPSHOT.jar
- open the solution
  - http://localhost:8080
